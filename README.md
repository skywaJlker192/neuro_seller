## 📖 О проекте

**NeuroSeller** — это универсальный Telegram-бот с искусственным интеллектом, который работает как виртуальный менеджер для вашего бизнеса. Бот консультирует клиентов, помогает выбрать товар или услугу, собирает контактные данные и автоматически передаёт готовые заявки реальному менеджеру.

### 🎯 Зачем это нужно?

- **Работает 24/7** — бот не спит, не болеет и не уходит в отпуск
- **Мгновенные ответы** — клиент не ждёт, пока менеджер освободится
- **Автоматический сбор заявок** — все данные структурированы и готовы к работе
- **Умные ответы** — YandexGPT понимает контекст и отвечает естественно
- **Универсальность** — один бот работает для магазина, салона красоты или автосервиса

##  Возможности

### 🛍️ Каталог товаров и услуг

Бот показывает категории и товары/услуги в виде удобных кнопок. Клиент просто нажимает на то, что ему интересно.

###  Умная форма заявки

При выборе товара или услуги бот запускает пошаговую форму:
1. **Имя** — как обращаться к клиенту
2. **Телефон** — для связи
3. **Бюджет** — для понимания возможностей

### 🤖 AI-консультант

На базе **YandexGPT** бот:
- Отвечает на вопросы о товарах и услугах
- Помогает с выбором
- Консультирует по ценам и характеристикам
- Отвечает естественно, как живой менеджер

### 🛡️ Защита от манипуляций

Бот **не даёт скидок** и **не меняет цены**. При попытках давления отвечает вежливо, но твёрдо, и предлагает передать вопрос менеджеру.

### 📊 Интеграции

- **Telegram-уведомления** — менеджер мгновенно получает новые заявки
- **Google Sheets** — все лиды автоматически сохраняются в таблицу для учёта и аналитики
- **База данных** — история диалогов и заявки хранятся локально

## 🚀 Как это работает

```
Клиент открывает бота
        ↓
Видит приветствие и категории
        ↓
Выбирает категорию (кнопка)
        ↓
Видит список товаров/услуг (кнопки)
        ↓
Нажимает на нужный товар/услугу
        ↓
Заполняет форму (имя → телефон → бюджет)
        ↓
✅ Заявка оформлена!
        ↓
📩 Менеджер получает уведомление в Telegram

```

## ⚙️ Настройка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка окружения

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_бота_от_BotFather
YANDEX_API_KEY=ваш_ключ_YandexGPT
YANDEX_FOLDER_ID=ваш_folder_id
MANAGER_CHAT_ID=chat_id_менеджера
PROXY_URL=http://proxy:port
GOOGLE_SHEETS_ID=id_таблицы
GOOGLE_CREDENTIALS_FILE=credentials.json
```

### 3. Получение токена бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен в `.env`

### 4. Настройка YandexGPT

1. Зайдите в [Yandex Cloud](https://cloud.yandex.ru/)
2. Создайте проект и получите API-ключ
3. Создайте каталог (folder) и получите folder_id
4. Включите YandexGPT API

### 5. Настройка Google Sheets

1. Создайте Google-таблицу для лидов
2. Получите ID таблицы из URL
3. Скачайте `credentials.json` из Google Cloud Console
4. Положите файл в корень проекта

### 6. Запуск бота

```bash
python -m app.main
```

## 📊 Как работают заявки

### Создание лида

Лид создаётся автоматически когда клиент заполняет форму. Данные сохраняются:
- В локальной базе данных (SQLite)
- В Google Sheets (если настроено)
- Отправляются менеджеру в Telegram

### Формат уведомления менеджеру

```
🔥 НОВЫЙ ЛИД!

📋 Ниша: Beauty Studio

Данные клиента:
• Имя: Иван
• Услуга: Мужская стрижка
• Контакт: +79991234567
• Бюджет: 2000
```

### Поля лида

| Поле | Описание |
|------|----------|
| `name` | Имя клиента |
| `interest` | Выбранный товар/услуга |
| `contact` | Телефон или email |
| `budget` | Бюджет клиента |
| `tg_user_id` | Telegram ID для связи |

## 🛡️ Безопасность и защита

### Защита от манипуляций

Бот запрограммирован:
- **Не давать скидки** — это прерогатива менеджера
- **Не менять цены** — все цены фиксированы в конфиге
- **Вежливо отказывать** при эмоциональном давлении
- **Предлагать передать вопрос менеджеру** в сложных ситуациях

### Пример ответа на просьбу о скидке:

> "К сожалению, я не могу предоставить скидку. Это решение принимает менеджер. Хотите, чтобы я передал ваш вопрос менеджеру? Он свяжется с вами и обсудит условия."

##  Решение проблем

### Бот не отвечает
- Проверьте токен в `.env`
- Убедитесь что прокси работает (если используется)
- Проверьте логи в консоли

### Лиды не создаются
- Проверьте что все поля формы заполнены
- Убедитесь что Google Sheets настроен
- Проверьте `MANAGER_CHAT_ID`

### Ошибка YandexGPT
- Проверьте API-ключ и folder_id
- Убедитесь что баланс Yandex Cloud не нулевой
- Бот автоматически использует запасные ответы при ошибках
